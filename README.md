# @squirrel-forge/node-taskrunner

Task runner and task class for sequential and parallel processing.

## Installation

```
npm i @squirrel-forge/node-taskrunner

```

## Usage

```
const { TaskRunner, Task } = require( '@squirrel-forge/node-taskrunner' );
```

 - Task
